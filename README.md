# sbtf
Standby Taskforce - Supporting disaster activations using simple front end 'spatial' tools and volunteers to address critical needs during times of disaster.

## Presentation at International Disaster conference
*Hurricane Dorian and beyond 
- ESRI web maps fed by Google Sheets (add on added by ESRI developers on our request within a day). - Silk (bought by Palantir) to create immersive dashboards that were featured by them. https://web.archive.org/web/20170519164711/https://www.silk.co/
- Google Sheets for collecting data by hundreds of volunteers with automated QA/QC and seperation of vetted, public and organisation sandboxed data
- Who, What, Where analysis
- Facebook, Twitter sentiment analysis
- Obligue imagery - disaster information collection pipeline
- Tasking manager based on HotOSM
- https://tinyurl.com/sbtf-geodm
